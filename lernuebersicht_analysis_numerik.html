Block 1 Taylor-Reihen
Was ist eine Taylor-Reihe? – Die Grundidee

Komplizierte Funktionen (z.B. sin(x), e^x) sind schwer direkt zu berechnen. Die Idee: Wir ersetzen sie durch ein Polynom (Summe von Potenzen von x), das an einem Punkt beliebig genau passt.

Das Taylor-Polynom der Ordnung n um den Entwicklungspunkt x₀ ist das Polynom, das dieselben Funktionswerte und ersten n Ableitungen wie f im Punkt x₀ hat.

Das Taylor-Polynom

Definition: Taylor-Polynom n-ter Ordnung um x₀
T_n(x) = Σ(k=0 bis n) [f⁽ᵏ⁾(x₀) / k!] · (x - x₀)^k = f(x₀) + f'(x₀)·(x-x₀) + f''(x₀)/2! · (x-x₀)² + f'''(x₀)/3! · (x-x₀)³ + ... + f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ

Merke: Der Koeffizient bei (x-x₀)ᵏ ist immer f⁽ᵏ⁾(x₀) / k! – also die k-te Ableitung an der Stelle x₀, dividiert durch k-Fakultät.

Wichtige Eigenschaft: T_n⁽ᵏ⁾(x₀) = f⁽ᵏ⁾(x₀) für k = 0, 1, ..., n. Das Polynom stimmt also mit f in allen Ableitungen bis zur n-ten überein.

Das Restglied (Fehler der Approximation)

Was ist das Restglied?

Das Restglied R_n(x) ist der Fehler, den wir machen, wenn wir f(x) durch T_n(x) ersetzen. Es gilt:

f(x) = T_n(x) + R_n(x)
Lagrange'sche Restglied-Darstellung
R_n(x) = f⁽ⁿ⁺¹⁾(ξ) / (n+1)! · (x - x₀)ⁿ⁺¹ wobei ξ ∈ [x₀, x] ein (unbekannter) Zwischenpunkt ist.

Da ξ unbekannt ist, benutzt man die obere Schranke: Man sucht das Maximum von |f⁽ⁿ⁺¹⁾| auf dem Intervall und schätzt damit ab.

Die Taylor-Reihe (unendliche Summe)

Definition: Taylor-Reihe

Wenn f unendlich oft differenzierbar ist, heißt die unendliche Summe:

Σ(k=0 bis ∞) f⁽ᵏ⁾(x₀)/k! · (x-x₀)ᵏ

die Taylor-Reihe von f um x₀. Für x₀ = 0 heißt sie Maclaurin-Reihe.

⚠️Die Taylor-Reihe konvergiert nicht automatisch gegen f(x). Die Konvergenz muss über das Restglied geprüft werden: R_n(x) → 0 für n → ∞?

Wichtige bekannte Taylor-Reihen (auswendig lernen!)

Standard-Reihen um x₀ = 0
eˣ = 1 + x + x²/2! + x³/3! + x⁴/4! + ... = Σ xⁿ/n! sin(x) = x - x³/3! + x⁵/5! - ... = Σ (-1)ⁿ x²ⁿ⁺¹/(2n+1)! cos(x) = 1 - x²/2! + x⁴/4! - ... = Σ (-1)ⁿ x²ⁿ/(2n)! ln(1+x) = x - x²/2 + x³/3 - ... = Σ (-1)ⁿ⁺¹ xⁿ/n, |x| < 1

Schritt-für-Schritt: Taylor-Polynom direkt berechnen

Methode A: Direkte Berechnung über Ableitungen
1
Ableitungen berechnen

Berechne f⁽⁰⁾(x), f⁽¹⁾(x), f⁽²⁾(x), f⁽³⁾(x), ... (bis zur gewünschten Ordnung n)

2
An der Stelle x₀ auswerten

Setze x = x₀ in alle Ableitungen ein → f⁽⁰⁾(x₀), f⁽¹⁾(x₀), ...

3
Formel einsetzen

T_n(x) = Σ f⁽ᵏ⁾(x₀)/k! · (x-x₀)ᵏ. Jeden Term ausschreiben und vereinfachen.

4
Allgemeines Glied angeben (optional)

Falls ein Muster erkennbar ist, gib die kompakte Summenformel an.

Methode B: Bekannte Reihe verwenden (schneller!)
1
Funktion umschreiben

Versuche f(x) als Transformation einer Standardfunktion zu erkennen. z.B. f(x) = x·e^(x-1) = x·e^u mit u = x-1

2
Bekannte Reihe substituieren

Ersetze u in der bekannten Reihe, z.B. e^u = Σ uⁿ/n!

3
Ausmultiplizieren

Multipliziere mit dem verbleibenden Faktor (z.B. x) und fasse zusammen.

Fehlerabschätzung – Schritt für Schritt

Schritt-für-Schritt: Fehler abschätzen
1
Restglied-Formel aufschreiben

|R_n(x)| = |f⁽ⁿ⁺¹⁾(ξ)| / (n+1)! · |x-x₀|^(n+1)

2
Maximum der (n+1)-ten Ableitung finden

Suche M = max|f⁽ⁿ⁺¹⁾(ξ)| auf dem Intervall [x₀, x]. Für e^x z.B. ist das e^x, also M ≤ e^|x|.

3
Schranke berechnen

|R_n(x)| ≤ M / (n+1)! · |x-x₀|^(n+1). Wert einsetzen → Zahl.

Vollständig gelöstes Beispiel (Übungsblatt 1, Aufgabe 2)

Aufgabe 1.2 cos(x) bis Ordnung n=4, Fehler für x=0.8 ▶

Gegeben: f(x) = cos(x), x₀ = 0, n = 4, x = 0.8

Vollständige Lösung
1
Ableitungen berechnen

f⁽⁰⁾(x) = cos(x) → f(0) = 1
f⁽¹⁾(x) = -sin(x) → f'(0) = 0
f⁽²⁾(x) = -cos(x) → f''(0) = -1
f⁽³⁾(x) = sin(x) → f'''(0) = 0
f⁽⁴⁾(x) = cos(x) → f⁽⁴⁾(0) = 1

2
Taylor-Polynom aufschreiben
T₄(x) = 1 + 0·x + (-1)/2! · x² + 0·x³ + 1/4! · x⁴ = 1 - x²/2 + x⁴/24
3
Fehlerabschätzung für x = 0.8

Das Restglied hat Ordnung n+1 = 5:

R₄(x) = f⁽⁵⁾(ξ)/5! · x⁵ = -sin(ξ)/120 · x⁵

Da |sin(ξ)| ≤ 1 für alle ξ:

|R₄(0.8)| ≤ 1/120 · (0.8)⁵ = 0.32768/120 ≈ 0.00273

Der Fehler ist kleiner als 0.003.

Aufgabe 1.1 f(x) = x·exp(x−1) um x₀ = 1 (beide Methoden) ▶

Methode A – direkte Ableitungen:

f(x) = x·e^(x-1). Produktregel: f'(x) = e^(x-1) + x·e^(x-1) = (1+x)e^(x-1)

f''(x) = e^(x-1) + (1+x)e^(x-1) = (2+x)e^(x-1)

Muster: f⁽ⁿ⁾(x) = (n+x)·e^(x-1)

An x₀ = 1: f⁽ⁿ⁾(1) = (n+1)·e⁰ = n+1

T_n(x) = Σ(k=0 bis n) (k+1)/k! · (x-1)^k

Methode B – bekannte Reihe:

Schreibe u = x-1, also f = (u+1)·e^u. Dann: e^u = Σ uⁿ/n!
f = u·e^u + e^u = Σ uⁿ⁺¹/n! + Σ uⁿ/n!

Zusammenführen nach Potenzen von u = (x-1) ergibt dasselbe Ergebnis.

Block 2 Skalare Funktionen über ℝⁿ
Was sind skalare Funktionen über ℝⁿ?

Eine skalare Funktion über ℝⁿ nimmt n Eingabewerte (einen n-dimensionalen Vektor) und liefert eine einzige reelle Zahl als Ausgabe:

f : ℝⁿ → ℝ, (x₁, x₂, ..., xₙ) ↦ f(x₁, x₂, ..., xₙ)

Beispiele:

  • Temperatur T(x, y, z, t) – Ort + Zeit → Temperaturwert
  • Verlustfunktion L(β₁, β₂) – Parameter → Fehlerwert
  • Volumen V(h, r) = π·r²·h – Höhe + Radius → Volumen

Niveaulinien

Was sind Niveaulinien?

Die Niveaulinie N_c einer Funktion f(x,y) zur Höhe c ist die Menge aller Punkte (x,y), für die f(x,y) = c gilt. Analog zu Höhenlinien auf einer Landkarte.

Nₓ = { (x,y) ∈ ℝ² | f(x,y) = c }
Aufgabe 2.1 Niveaulinien von f(x,y) = x·y skizzieren ▶

Niveaulinie N_c: x·y = c → y = c/x (Hyperbeln für c ≠ 0)

  • N₀: x·y = 0 → x = 0 oder y = 0 (die Achsen)
  • N₁: y = 1/x (gleichseitige Hyperbel, 1. und 3. Quadrant)
  • N₋₁: y = -1/x (2. und 4. Quadrant)
  • N₂: y = 2/x (flachere Hyperbeln)

Die Niveaulinien sind also Hyperbeln mit der x- und y-Achse als Asymptoten.

Block 3 Partielle Ableitungen · Gradient · Linearisierung
Was ist eine partielle Ableitung?

Die partielle Ableitung nach x₁ gibt die lokale Änderung von f, wenn nur x₁ variiert wird, während alle anderen Variablen konstant gehalten werden.

∂f/∂xᵢ = lim(h→0) [f(..., xᵢ+h, ...) - f(..., xᵢ, ...)] / h

Schreibweisen: ∂f/∂x oder ∂_x f oder f_x

Rechenregel: Partielle Ableitung bilden

So rechnet man partielle Ableitungen
1
Variablen aussuchen

Entscheide, nach welcher Variable du ableiten willst (z.B. nach x).

2
Alle anderen als Konstante behandeln

Alle anderen Variablen (y, z, ...) werden wie Zahlen behandelt. Nach y differenzieren? Dann x konstant!

3
Normale Ableitungsregeln anwenden

Produkt-, Ketten-, Quotientenregel – alles normal, nur mit der "freigegebenen" Variable.

Beispiel: f(x,y) = x²y + xe^y
∂f/∂x = 2xy + e^y (y wird als Konstante behandelt) ∂f/∂y = x² + xe^y (x wird als Konstante behandelt) ∂²f/∂x² = 2y (nochmal nach x von ∂f/∂x) ∂²f/∂y² = xe^y (nochmal nach y von ∂f/∂y) ∂²f/∂x∂y = 2x + e^y (∂f/∂x nach y ableiten)

Satz von Schwarz: Bei ausreichend glatten Funktionen gilt ∂²f/∂x∂y = ∂²f/∂y∂x – die Reihenfolge spielt keine Rolle.

Der Gradient

Was ist der Gradient?

Der Gradient ∇f einer skalaren Funktion f : ℝⁿ → ℝ ist ein Zeilenvektor aller ersten partiellen Ableitungen:

∇f = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ)

Er zeigt in die Richtung des stärksten Anstiegs der Funktion.

Die Hesse-Matrix

Was ist die Hesse-Matrix?

Die Hesse-Matrix Hf enthält alle zweiten partiellen Ableitungen – sie beschreibt die lokale Krümmung der Funktion:

Hf = ⎡ f_x₁x₁ f_x₁x₂ ... f_x₁xₙ ⎤ ⎢ f_x₂x₁ f_x₂x₂ ... f_x₂xₙ ⎥ ⎣ f_xₙx₁ f_xₙx₂ ... f_xₙxₙ ⎦

Die Hesse-Matrix ist symmetrisch (wegen Satz von Schwarz).

Linearisierung skalarer Funktionen

Was ist die Linearisierung?

Die Linearisierung ℓ ist die beste lineare (affine) Näherung von f in der Nähe eines Punktes (x̂, ŷ). Sie ist die Tangentialebene an den Funktionsgraphen.

Linearisierung in ℝ² (und allgemein in ℝⁿ)
ℓ(x,y) = ∇f(x̂,ŷ) · (x-x̂, y-ŷ)ᵀ + f(x̂,ŷ) = f_x(x̂,ŷ)·(x-x̂) + f_y(x̂,ŷ)·(y-ŷ) + f(x̂,ŷ) Allgemein: ℓ(x) = ∇f(x̂) · (x - x̂) + f(x̂)

Schritt-für-Schritt: Linearisierung berechnen

Vorgehen bei der Linearisierung
1
Gradient ∇f berechnen

Alle partiellen Ableitungen ∂f/∂xᵢ berechnen.

2
Gradient an Punkt (x̂, ŷ) auswerten

Werte ∇f an der Entwicklungsstelle aus → Zahlenwerte.

3
Funktionswert f(x̂, ŷ) berechnen

Den Wert der Funktion selbst an der Stelle auswerten.

4
Formel einsetzen

ℓ(x,y) = f_x(x̂,ŷ)·(x-x̂) + f_y(x̂,ŷ)·(y-ŷ) + f(x̂,ŷ)

5
Abweichung prüfen (falls gefragt)

|f(x₀,y₀) - ℓ(x₀,y₀)| ausrechnen.

Aufgabe 2.4b / 2.6 g(x,y) = e^(−5x+2y) linearisieren bei P:(2,5) ▶
Vollständige Lösung
1
Partielle Ableitungen
∂g/∂x = -5 · e^(-5x+2y) ∂g/∂y = 2 · e^(-5x+2y)
2
Auswertung bei (2,5)

Exponent: -5·2 + 2·5 = -10+10 = 0, also e⁰ = 1

g(2,5) = 1, ∂g/∂x|(2,5) = -5, ∂g/∂y|(2,5) = 2
3
Linearisierung
ℓ(x,y) = -5·(x-2) + 2·(y-5) + 1 = -5x + 10 + 2y - 10 + 1 = -5x + 2y + 1
4
Abweichung bei Q:(2.1, 4.9)

g(2.1, 4.9) = e^(-5·2.1+2·4.9) = e^(-10.5+9.8) = e^(-0.7) ≈ 0.4966
ℓ(2.1, 4.9) = -5·2.1 + 2·4.9 + 1 = -10.5 + 9.8 + 1 = 0.3
Abweichung: |0.4966 - 0.3| ≈ 0.197

Richtungsableitung

Was ist die Richtungsableitung?

Die Richtungsableitung gibt an, wie schnell sich f in einer beliebigen Richtung v ändert (nicht nur entlang der Koordinatenachsen).

∂f/∂v (x) = ∇f(x) · v/‖v‖ = ∇f(x) · v̂

Man projiziert also den Gradienten auf den normierten Richtungsvektor.

Richtungsableitung berechnen
1
Gradient berechnen und auswerten

∇f an der gewünschten Stelle berechnen.

2
Richtungsvektor normieren

v̂ = v / ‖v‖. Norm: ‖v‖ = √(v₁² + v₂²)

3
Skalarprodukt berechnen

∂f/∂v = ∇f · v̂ = f_x · v̂₁ + f_y · v̂₂

Block 4 Optimierung & Lagrange-Multiplikatoren

Extremalsuche ohne Nebenbedingungen

Kritischer Punkt

Ein Punkt P = (x₁,...,xₙ) heißt kritischer Punkt, wenn der Gradient Null ist: ∇f(P) = 0 (alle partiellen Ableitungen = 0).

Kritische Punkte sind Kandidaten für Extrema (Minima, Maxima) oder Sattelpunkte.

Hesse-Matrix zur Klassifikation

Satz: Bedingungen für Extrema (Eigenwerte der Hesse-Matrix)
Sei P ein kritischer Punkt (∇f(P) = 0): Alle Eigenwerte von Hf(P) > 0 → lokales MINIMUM Alle Eigenwerte von Hf(P) < 0 → lokales MAXIMUM Eigenwerte haben verschiedene Vorzeichen → SATTELPUNKT Für ℝ² einfacher: Determinante det(Hf) = f_xx·f_yy - (f_xy)² det > 0 und f_xx > 0 → Minimum det > 0 und f_xx < 0 → Maximum det < 0 → Sattelpunkt det = 0 → keine Aussage
Schritt-für-Schritt: Extrema ohne Nebenbedingung
1
Gradienten = 0 setzen

∂f/∂x = 0 und ∂f/∂y = 0 → Gleichungssystem lösen → kritische Punkte

2
Hesse-Matrix aufstellen

Alle zweiten partiellen Ableitungen berechnen.

3
Hesse-Matrix auswerten und klassifizieren

Determinante und f_xx berechnen → Kriterium anwenden.

Aufgabe 3.2 f(x,y) = −x³ + 6xy − 3y² + 12 auf Extrema untersuchen ▶
Vollständige Lösung
1
Kritische Punkte
∂f/∂x = -3x² + 6y = 0 → y = x²/2 ∂f/∂y = 6x - 6y = 0 → y = x

Einsetzen: x = x²/2 → x² - 2x = 0 → x(x-2) = 0

Kritische Punkte: (0,0) und (2,2)

2
Hesse-Matrix
f_xx = -6x, f_yy = -6, f_xy = 6 Hf = ⎡-6x 6⎤ ⎣ 6 -6⎦ det(Hf) = (-6x)·(-6) - 6² = 36x - 36
3
Klassifikation

Punkt (0,0): det = 36·0 - 36 = -36 < 0 → Sattelpunkt

Punkt (2,2): det = 36·2 - 36 = 36 > 0, f_xx = -12 < 0 → lokales Maximum

f(2,2) = -8 + 24 - 12 + 12 = 16

Extremalsuche mit Nebenbedingungen: Lagrange-Multiplikatoren

Was sind Lagrange-Multiplikatoren?

Gesucht: Extremum von f(x) unter der Nebenbedingung g(x) = 0.

Idee: An einem Extremum sind die Gradienten von f und g parallel: ∇f = -s · ∇g (wobei s der Lagrange-Multiplikator ist).

Man fasst beides in der Lagrange-Funktion zusammen:

ℒ(x₁,...,xₙ, s) = f(x₁,...,xₙ) + s · g(x₁,...,xₙ)
Schritt-für-Schritt: Lagrange-Methode
1
Nebenbedingung auf Nullform bringen

g(x,y) = 0 aufschreiben (z.B. xyz - V = 0)

2
Lagrange-Funktion aufstellen

ℒ(x,y,s) = f(x,y) + s·g(x,y)

3
Alle partiellen Ableitungen = 0 setzen

∂ℒ/∂x = 0, ∂ℒ/∂y = 0, ∂ℒ/∂s = 0 (das ergibt automatisch die Nebenbedingung)

4
Gleichungssystem lösen

Oft: s aus zwei Gleichungen berechnen und gleichsetzen → x,y,z aus Symmetrie oder Einsetzen

5
Optimale Werte einsetzen und prüfen

Plausibilitätsprüfung: Handelt es sich wirklich um ein Minimum/Maximum?

Aufgabe 4.3 Cobb-Douglas f(L,K)=L^0.5·K^0.5 bei Budget 10L+20K=1000 ▶
Vollständige Lösung
1
Nebenbedingung

g(L,K) = 10L + 20K - 1000 = 0

2
Lagrange-Funktion
ℒ(L,K,s) = L^0.5 · K^0.5 + s·(10L + 20K - 1000)
3
Ableitungen = 0
∂ℒ/∂L = 0.5·L^(-0.5)·K^0.5 + 10s = 0 → 0.5·K^0.5/L^0.5 = -10s ∂ℒ/∂K = 0.5·L^0.5·K^(-0.5) + 20s = 0 → 0.5·L^0.5/K^0.5 = -20s ∂ℒ/∂s = 10L + 20K - 1000 = 0
4
s eliminieren

Erste Gleichung dividiert durch zweite:

K/L = 10s / (20s) · ... → K/L = 1/2 → K = L/2

Einsetzen in Nebenbedingung: 10L + 20·(L/2) = 1000 → 20L = 1000 → L = 50, K = 25

Block 5 Jacobi-Matrix & Differenzieren nach Vektoren/Matrizen
Vektorwertige Funktionen

Bisher: f : ℝⁿ → ℝ (skalare Ausgabe). Jetzt: f : ℝⁿ → ℝᵐ (Vektor-Ausgabe):

f(x) = [f₁(x), f₂(x), ..., f_m(x)]ᵀ
Was ist die Jacobi-Matrix?

Die Jacobi-Matrix J ist die Sammlung aller ersten partiellen Ableitungen einer vektorwertigen Funktion f : ℝⁿ → ℝᵐ. Sie ist eine m×n-Matrix:

J = ∇_x f = ⎡ ∂f₁/∂x₁ ∂f₁/∂x₂ ... ∂f₁/∂xₙ ⎤ ⎢ ∂f₂/∂x₁ ... ... ... ⎥ ⎣ ∂fₘ/∂x₁ ... ... ∂fₘ/∂xₙ ⎦ J(i,j) = ∂fᵢ/∂xⱼ

Dimensionen: m Zeilen (eine pro Ausgabe-Komponente), n Spalten (eine pro Eingabe-Variable)

Kettenregel für Gradienten

Kettenregel (mehrdimensional)
Wenn z = f(g(x)): ∂z/∂x = (∂f/∂g) · (∂g/∂x) = J_f · J_g Für f(z), z = g(x): ∇_x f = ∇_z f · J_g(x)

Wichtige Rechenregeln für Gradienten von Matrizen

Wichtige Identitäten (auswendig lernen!)
∂/∂X Xᵀa = aᵀ (a konstanter Vektor) ∂/∂X aᵀX = aᵀ (a konstanter Vektor) ∂/∂X Xᵀ B X = X(B + Bᵀ) (B symmetrisch: 2BX) ∂/∂X tr(AXB) = AᵀBᵀ (Spur-Ableitung) ∂/∂X f(X)ᵀ = (∂f/∂X)ᵀ
Aufgabe 5.4a f(z) = log(1+z), z = xᵀx – Gradient mit Kettenregel ▶
Kettenregel anwenden
1
Äußere Ableitung
∂f/∂z = 1/(1+z) → Dimension: (1×1)
2
Innere Ableitung
∂z/∂x = ∂(xᵀx)/∂x = 2xᵀ → Dimension: (1×D)
3
Kettenregel
∂f/∂x = (1/(1+xᵀx)) · 2xᵀ → Dimension: (1×D)
Block 6 Mehrfachintegrale
Was ist ein Doppelintegral?

Ein Doppelintegral integriert eine Funktion f(x,y) über einen zweidimensionalen Bereich D. Es entspricht dem Volumen unter der Fläche z = f(x,y) über dem Bereich D.

∬_D f(x,y) dA = ∬_D f(x,y) dx dy

Fubini'scher Satz: Reduktion auf iterierende Integrale

Fubini (Rechteckgebiet)
Für D = [a,b] × [c,d]: ∬_D f(x,y) dx dy = ∫_c^d [∫_a^b f(x,y) dx] dy = ∫_a^b [∫_c^d f(x,y) dy] dx
Schritt-für-Schritt: Doppelintegral berechnen
1
Integrationsreihenfolge festlegen

Wähle (innen über x, außen über y) oder umgekehrt. Bei Rechteckgebiet beliebig.

2
Inneres Integral berechnen

Integriere über die innere Variable. Die andere Variable wird dabei als Konstante behandelt.

3
Ergebnis als Funktion der äußeren Variable einsetzen

Das Ergebnis des inneren Integrals ist jetzt nur noch eine Funktion der äußeren Variable.

4
Äußeres Integral berechnen

Normales 1D-Integral auswerten.

Aufgabe 6.1a ∬ e^(x−y) dx dy, D = [0,2]×[0,1] ▶
Vollständige Lösung
1
Faktorisieren

e^(x-y) = e^x · e^(-y) → Separierbar! Das Integral zerfällt in zwei unabhängige Integrale.

2

∫_0^2 e^x dx = [e^x]_0^2 = e² - 1
3
Äußeres Integral (über y)
∫_0^1 e^(-y) dy = [-e^(-y)]_0^1 = -e^(-1) - (-1) = 1 - 1/e
4
Ergebnis
∬ = (e² - 1) · (1 - 1/e) = e² - e - 1 + 1/e ≈ (7.389-1)·(0.632) ≈ 4.036

Fehlertypen

Absoluter Fehler
e_abs = |exakter Wert − Näherung|

Der tatsächliche Abstand vom wahren Wert.

Relativer Fehler
e_rel = |exakter Wert − Näherung| / |exakter Wert|

Der Fehler in Relation zur Größe des Wertes.

Konditionszahl

Was ist die Konditionszahl?

Die Konditionszahl κ misst, wie stark sich ein kleiner Fehler in der Eingabe auf die Ausgabe auswirkt. Eine gut konditionierte Aufgabe (κ ≈ 1) verstärkt Eingabefehler kaum; eine schlecht konditionierte (κ ≫ 1) verstärkt sie stark.

κ(x) = |x · f'(x) / f(x)|
Beispiel: Konditionszahlen (Aufgabe 7.2)

x · y (Multiplikation)1 (immer gut)Keine Verstärkung von Fehlern
√x (Wurzel)1/2 (immer gut)Dämpft Fehler sogar

Auslöschung

⚠️ Auslöschung tritt auf, wenn zwei fast gleich große Zahlen subtrahiert werden. Die signifikanten Stellen heben sich auf und das Ergebnis hat kaum noch korrekte Stellen. Beispiel: 1.23456 - 1.23450 = 0.00006 (nur 1 korrekte Stelle trotz 6-stelliger Eingabe).

Gleitkommadarstellung

Was ist Gleitkomma-Arithmetik?

Computer speichern Zahlen im Format: ±1.m × 2^e mit begrenzter Mantisse m und begrenztem Exponent e. Dadurch sind nicht alle reellen Zahlen exakt darstellbar → Rundungsfehler.

2

3

Block 8 Newton-Verfahren
Was ist das Newton-Verfahren?

Das Newton-Verfahren ist ein iteratives Verfahren zur Nullstellenbestimmung von f(x) = 0. Es benutzt die Idee der Linearisierung: Man approximiert f durch ihre Tangente am aktuellen Punkt und findet deren Nullstelle als nächsten Startpunkt.

Newton-Iterationsformel
x_{n+1} = x_n - f(x_n) / f'(x_n)

Geometrische Intuition

Man zeichnet die Tangente an den Graphen von f im Punkt (xₙ, f(xₙ)). Die Nullstelle der Tangente ist der neue Näherungswert x_{n+1}.

Schritt-für-Schritt: Newton-Verfahren durchführen
1
f(x) und f'(x) berechnen

Die Ableitung muss bekannt sein (analytisch).

2
Startwert x₀ wählen

Möglichst nahe an der gesuchten Nullstelle. Regel von Fourier: x₀ mit f(x₀)·f''(x₀) > 0 wählen.

3
Iteration durchführen

x_{n+1} = x_n - f(x_n)/f'(x_n). Auf genug Stellen runden (6 Nachkommastellen).

4
Abbruchkriterium prüfen

|f(x_n)| < ε = 0.0001. Falls erfüllt: Stopp.

Fehlerabschätzungen

A-priori-Fehlerabschätzung (vor der Iteration)
|x_k - z| ≤ (2m/M) · q^(2^k) mit m = min|f'(x)| auf [a,b] M = max|f''(x)| auf [a,b] q = (M/(2m)) · |x₀ - z| ≤ (M/(2m)) · ρ (ρ = b-a konservativ)
A-posteriori-Fehlerabschätzung (nach Iteration)
|x_k - z| ≤ |f(x_k)| / m (scharf) |x_k - z| ≤ (M/(2m)) · |x_k - x_{k-1}|² (schwächer)
⚡ Konvergenzordnung: Das Newton-Verfahren hat quadratische Konvergenz – die Anzahl der korrekten Dezimalstellen verdoppelt sich mit jedem Schritt. Das bedeutet: q² in der a-priori-Schranke.

Block 9 Numerische Differentiation
Idee der numerischen Differentiation

Wenn f' nicht analytisch bekannt ist (oder sehr komplex), approximiert man sie durch Differenzenquotienten: kleine Schritte h anstelle von infinitesimalem Grenzübergang.

Differenzenquotienten

Zentrale Differenzenformeln (Ordnung O(h²))
f'(x₀) ≈ [f(x₀+h) - f(x₀-h)] / (2h) f''(x₀) ≈ [f(x₀+h) - 2f(x₀) + f(x₀-h)] / h²
Fehlerabschätzung (erster Ableitung, zentral)
|f'_h(x₀) - f'(x₀)| ≤ h²/6 · |f'''(x₀)| + O(h⁴)
⚠️ Auslöschungsproblem bei kleinem h: Je kleiner h, desto genauer die Approximation – aber bei sehr kleinem h subtrahiert man fast gleiche Zahlen → Auslöschung dominiert → Fehler wächst wieder! Es gibt ein optimales h.
Schritt-für-Schritt: Numerische Ableitung berechnen
1
Schrittweite h wählen

h = 1/2^k für k = 1, 2, ..., 5 (halbieren)

2
Funktionswerte berechnen

f(x₀+h) und f(x₀-h) exakt (oder mit Computerpräzision) auswerten.

3
Differenzenquotient berechnen

[f(x₀+h) - f(x₀-h)] / (2h) für f', bzw. [f(x₀+h) - 2f(x₀) + f(x₀-h)] / h² für f''

4
Fehler gegen exakten Wert berechnen

|Approximation - f'(x₀)| mit bekanntem analytischem Wert vergleichen.

5
Fehlerentwicklung analysieren

Wird der Fehler mit halbiertm h auf ~1/4 kleiner? → O(h²)-Konvergenz bestätigt.

Aufgabe 9.2b Auslöschung: f(x) = x³−x bei x₀ = 1/√3 (lokales Extremum) ▶

f'(x) = 3x²-1, an x₀=1/√3: f'(1/√3) = 3·(1/3)-1 = 0 (Extremum)

Das ist problematisch: f(x₀+h) ≈ f(x₀-h), denn x₀ ist ein Extremum. Bei der Differenz f(x₀+h) - f(x₀-h) heben sich die Hauptterme auf → fast nichts übrig.

Für kleines h gilt f(x₀+h) ≈ f(x₀-h) sehr genau, also ist die Differenz fast 0, aber durch 2h geteilt → hohe relative Fehler durch Auslöschung.

📌Empfehlung: Numerische Differentiation ist instabil, wenn f'(x₀) ≈ 0. In diesem Fall besser eine andere Methode (z.B. automatische Differentiation) verwenden oder den Auswertungspunkt leicht verschieben.
